# Listas API

### Requirements

- [Rust](https://www.rust-lang.org/tools/install)
- [NodeJS](https://nodejs.org)
- [MongoDB](https://docs.mongodb.com/manual/installation)


### Run

#### Dev
```bash
cargo watch -x 'run --bin listas'
```

### Migrations

#### Up
```bash
$ ./scripts/up.sh
```

#### Down
```bash
$ ./scripts/down.sh
```
